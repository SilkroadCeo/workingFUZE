# Icons Directory

## Required Icons

Please add the following icon files to this directory:

### iOS Icons (PNG format):
- `icon-76x76.png` - iPad
- `icon-120x120.png` - iPhone Retina
- `icon-152x152.png` - iPad Retina
- `icon-180x180.png` - iPhone 6 Plus & newer

### Android/PWA Icons (PNG format):
- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`

### Favicons (PNG format):
- `favicon-16x16.png`
- `favicon-32x32.png`

## How to Generate Icons

You can use online tools to generate all required icons from a single source image:

1. **PWA Builder Image Generator**: https://www.pwabuilder.com/imageGenerator
2. **Real Favicon Generator**: https://realfavicongenerator.net/

Simply upload your logo (recommended: 1024x1024 px PNG with transparent background) and download all generated icons.

## Source Image Requirements

- **Format**: PNG with transparent background
- **Size**: At least 1024x1024 px
- **Design**: Simple, recognizable icon that works at small sizes
- **Colors**: Should match your brand (#ff6b9d pink primary color)

## Quick Tip

For the Muji dating app, consider using:
- The letter "M" in a stylized font
- A heart icon
- A combination of both

Make sure the icon is clearly visible on both light and dark backgrounds.
