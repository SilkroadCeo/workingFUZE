# Настройка кнопки Telegram бота

## Описание

На сайте добавлена плавающая кнопка для связи с Telegram ботом, аналогичная TON Dating. Кнопка отображается в правом нижнем углу и анимирована.

## Как настроить

### Шаг 1: Узнайте имя вашего бота

1. Откройте Telegram и найдите @BotFather
2. Отправьте команду `/mybots`
3. Выберите вашего бота
4. Нажмите "Bot Settings" → "Edit Bot Settings"
5. Скопируйте **username** бота (например: `my_support_bot`)

**Или через API:**
```bash
curl "https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getMe"
```
В ответе найдите поле `"username"`

### Шаг 2: Обновите конфигурацию

Откройте файл `frontend/index.html` и найдите строку (примерно строка 5030):

```javascript
const TELEGRAM_BOT_USERNAME = 'YOUR_BOT_USERNAME'; // Замените на имя вашего бота
```

Замените `YOUR_BOT_USERNAME` на имя вашего бота:

```javascript
const TELEGRAM_BOT_USERNAME = 'my_support_bot'; // Ваш бот
```

### Шаг 3: Сохраните и перезапустите

Сохраните файл и перезапустите приложение.

## Как это работает

### В Telegram Mini App
Когда пользователь нажимает кнопку внутри Telegram:
- Открывается чат с ботом внутри Telegram
- Используется `window.Telegram.WebApp.openTelegramLink()`

### В обычном браузере
Когда пользователь нажимает кнопку в браузере:
- Открывается новая вкладка с `https://t.me/your_bot`
- Пользователь может начать чат с ботом

## Позиционирование

### Десктоп:
- Правый нижний угол
- 20px от края
- 60x60 пикселей

### Мобильные:
- Правый нижний угол
- 16px от края
- 56x56 пикселей
- Поднята выше (80px снизу) чтобы не мешать навигации

## Кастомизация

### Изменить текст tooltip:
```html
<button class="telegram-bot-btn" onclick="openTelegramBot()" data-tooltip="Ваш текст">
```

### Изменить цвет:
В CSS найдите `.telegram-bot-btn` и измените:
```css
background: linear-gradient(135deg, #0088cc, #00a0dc); /* Ваши цвета */
```

### Изменить позицию:
```css
.telegram-bot-btn {
    bottom: 20px;  /* Расстояние снизу */
    right: 20px;   /* Расстояние справа */
}
```

## Troubleshooting

### Кнопка не видна
- Проверьте z-index (должен быть 9999)
- Проверьте что нет других элементов поверх

### Кнопка не работает
- Убедитесь что username бота указан правильно
- Проверьте консоль браузера на ошибки

### Неправильная ссылка
- Убедитесь что `TELEGRAM_BOT_USERNAME` указан БЕЗ @
- Правильно: `my_bot`
- Неправильно: `@my_bot`
